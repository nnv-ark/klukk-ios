# KLUKK! 1.1 — App Store update copy

## What's New (paste into the 1.1 version's release notes)

```
KLUKK! goes dark. 🟤

• Dark mode — a warm, earthy brown canvas after sundown. The pink button
  stays exactly as pink as you remember it.
• One clear calendar button — the destination chip and your session count
  now live in a single pill at the bottom. Less clutter, same control.
• Honest feedback — if a .xml export ever fails, KLUKK now says so instead
  of pretending it worked.
• Better with VoiceOver — the calendar button now describes exactly what it
  will do.
• Under-the-hood polish for faster, smoother updates to come.

Tap. Time. It's in your calendar.
```
